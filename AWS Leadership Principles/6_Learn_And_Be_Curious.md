# LP - Learn and Be Curious

# Story - Personal Agentic AI Chatbot on Personal Website

## Situation:  

>As an SA, I could see GenAI and agentic systems rapidly becoming central to how ISVs design customer experiences — especially around retrieval‑augmented assistants, tool‑calling workflows, and autonomous agents. I didn’t want to rely on surface‑level understanding or wait for a project to force me into the space. I wanted deep, practical knowledge that would let me guide customers confidently as these patterns mature.

## Task:  

>I set myself a personal challenge: build a fully agentic, RAG‑enabled assistant for my own website. The goal was intentionally ambitious — the agent should answer any question about me (profile, skills, certifications, projects, blogs) and also take actions through tools like sending me an email for unanswered queries or performing external lookups like Wikipedia search. This wasn’t assigned or tied to deliverables; it was a deliberate way to push myself into the details of agentic architecture rather than staying at the conceptual level.

## Action:  

>To build a strong foundation, I enrolled in Morgan Willis’s Strands Agents course — taught by an AWS Developer Advocate — and immediately applied each concept to my project. Even though the system isn’t fully deployed yet, the hands‑on work has already forced me into real architectural decisions:

> - how to chunk and embed my content so retrieval stays grounded and avoids hallucinations
> - how to design the agent’s toolset so it can choose between RAG retrieval, email, or web search based on intent
> - how to implement guardrails so action tools like email can’t be misused or triggered incorrectly
> - how to interpret reasoning traces to understand why the agent selects certain tools


## Result:  

>The project is still in progress, but it has already given me a deep, practical understanding of how agentic systems work in real architectures — retrieval grounding, tool arbitration, and safe action execution. It’s fundamentally changed how I think about designing GenAI solutions for customers, and it’s given me current, hands‑on expertise in exactly the area where SA conversations — and my interview loop — are evolving.
